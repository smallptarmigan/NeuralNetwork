# Neural Network

環境初期設定

```bash
python3 -m venv venv # 環境生成
pip install --upgrade pip # pipの更新
```

環境保存と復元

```bash
 pip freeze > requirements.txt # 保存
pip install -r requirements.txt # 復元
```

仮想環境のアクティブ化と非アクティブ化

```bash
source venv/bin/activate #アクティブ化
deactivate # 非アクティブ化
```
